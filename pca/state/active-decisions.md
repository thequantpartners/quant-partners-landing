# Active Decisions

- **ICP Objetivo:** Firmas de abogados de inmigración especializadas en EB-2 NIW / O-1.
- **Ángulo de Ventas:** "Self-Liquidating Funnel". El sistema pre-califica a los leads y les cobra una consulta inicial ($150-$300) antes de hablar con el abogado. Esto genera un ROI inmediato para la firma y justifica el setup de $2.5k y retainer mensual de $1.5k de TQP.
- **Diseño de Landing Page:** Estilo B2B High-Ticket. Tonos oscuros (Navy/Black) con acentos dorados (Gold). Uso de tipografía monoespaciada para la sección de "Unit Economics" (simulando un terminal financiero).
